Setting up a Syntatica development environment.

Download and install Java 1.5 or 1.6

Download and install eclipse J2EE

Launch eclipse and add the plug-ins

·      Subclipse

·      AspectJ

·      Equinox AspectJ

·      Maven2 (m2eclipse)

·      UIMA

Install the subclipse plug-in for eclipse.

Select Help->Software Updates…

Click on ‘Add Site…’

Enter:

http://subclipse.tigris.org/update_1.4.x

 

http://download.eclipse.org/tools/ajdt/34/dev/update

Install Equinox AspectJ plug-in for eclipse

http://download.eclipse.org/tools/ajdt/34/dev/equinox-aspects

Install the Maven2 plug-in for eclipse.

http://m2eclipse.sonatype.org/update/

Install the UIMA plug-in for eclipse.

http://www.apache.org/dist/incubator/uima/eclipse-update-site

Checkout the UIMA core and sandbox from the UIMA SVN repositories.

Open the ‘SVN Repository Exploring’ perspective.

 

 

Select Add SVN Repository

Enter http://svn.apache.org/repos/asf/incubator/uima/uimaj/trunk

Click on Finish

Select Add SVN Repository

Enter http://svn.apache.org/repos/asf/incubator/uima/sandbox/trunk

Click on Finish

 

 

 

 Expand the view of the two repositories.

Select all of the projects within each repository.  Right click on the projects and select ‘Checkout…’.

Click on Finish

Compile the necessary projects

 

 

cd uimaj

mvn install

cd ../uimaj-core

mvn install

cd../PearPackagingMavenPlugin

mvn install:install-file -DgroupId=org.apache.uima -DartifactId=uimaj-core -Dversion=2.2.2-incubating -Dpackaging=jar -Dfile=../uimaj-core/target/uima-core.jar

mvn install

Answer ‘y’ to all questions posed.

cd ../WhitespaceTokenizer

mvn install:install-file -DgroupId=org.apache.uima -DartifactId=PearPackagingMavenPlugin -Dversion=2.2.2-incubating -Dpackaging=jar -Dfile=../PearPackagingMavenPlugin/target/uima-pear-maven-plugin.jar

mvn install

cd ../Tagger

mvn install:install-file -DgroupId=org.apache.uima -DartifactId=WhitespaceTokenizer -Dversion=2.2.2-incubating -Dpackaging=jar -Dfile=../WhitespaceTokenizer/target/uima-an-wst.jar

mvn install

cd../SimpleServer

mvn install

Create a simple web application using the above components.

 

Create a project for a webapp to assemble the above compiled modules and create a sample web application to run the UIMA pipes against.

 

Switch to the Java perspective

 

Select File->New->Project…

 

Select Maven->Maven Project

Click Next

 

Click on the Advanced link and unselect Separate projects for modules.

Click Next

From the Catalog selector, click on Internal

Select the entry in the list for the artifact maven-archetype-webapp

Click Next

Enter the group id com.syntatica.uima

The artifact id is the eclipse project name.  The example is parts-of-speech

Click Finish

In the generated pom.xml file in the project, replace the dependencies tag with the following:

<dependencies>

      <artifactId>junit</artifactId>

Your environment is now set up.  Please refer to the Syntatica UIMA Workflow configuration document to configure the processing workflow.
